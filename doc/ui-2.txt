#Change OK and NG Icon.

You can change OK and NG icons on icon selector dialog.


| Icon type | description |
|-----------|-------------|
| OK        | OK icon will is showed when you tap completely safe button. |
| NG        | NG icon cause the playing game over.                        |



You can change each icon with icon type selector in icon selector dialog.

![icon type selector](img/icon-selector-ng.png)

to display above modal dialog, you tap wrench mark
<i class="fas fa-wrench"></i>
at left upper.
